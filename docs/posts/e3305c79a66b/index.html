Optimizer: 优化器

SGD（Stochastic gradient descent）

(Chinese Name: 随机梯度下降法)
The main idea of SGD can be explained by the following formula:

W←W−η∂L∂WW \leftarrow W - \eta \frac{\partial L}{\partial W} W←W−η∂W∂L​

where η\etaη stands for learning rate, which is ussally set to be predefined values like 0.001 or 0.01.

SGD is easy to complement, but sucks at its efficiency. In an example of function f(x,y)=120x2+y2f(x,y)= \frac{1}{20}x^2+ y^2f(x,y)=201​x2+y2
the gradient of the function is nearly parallel to the y axis, which means it can be hard to find the “best position” of the function
