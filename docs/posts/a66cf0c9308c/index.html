fork别的仓库

下面是deepseek对于fork的描述：

  • Fork 的仓库是你的独立副本，修改不会影响原仓库。

  • Fork 的仓库不会自动同步原仓库的更新，需要手动同步。

  • 你可以通过命令行或网页端将原仓库的更新同步到你的 Fork 仓库。

  • 如果你想贡献代码，可以通过 Pull Request 提交更改。
